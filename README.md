# SEO Audit API

A TypeScript backend API that analyzes a website URL and generates a technical SEO audit report.

## Goal

This project is built as an internship preparation project for a digital marketing agency. It focuses on technical SEO checks that help marketing and development teams identify website issues.

## Planned Features

- URL validation
- HTML fetching
- Title and meta description analysis
- Heading structure analysis
- Image alt text checks
- Canonical and viewport checks
- robots.txt and sitemap.xml checks
- SEO score calculation
- Actionable recommendations
- Audit history with Prisma and SQLite

## Tech Stack

- Node.js
- TypeScript
- Express
- Axios
- Cheerio
- Zod
- Prisma
- SQLite
- dotenv
- cors
